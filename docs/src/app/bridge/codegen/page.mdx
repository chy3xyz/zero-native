# Type-Safe IPC (Codegen)

zero-native ships with a comptime TypeScript declaration generator that produces typed `.d.ts` files from Zig command schemas. The generator is a pure Zig module — no external tooling, no extra build step beyond calling a function and writing the result.

## Schemas

```zig
const codegen = @import("zero-native").bridge.codegen;

pub const CommandSchema = struct {
    name: []const u8,
    params: []const ParamSchema,
    /// JSON type name: "void", "string", "number", "object"
    result: []const u8,
};

pub const ParamSchema = struct {
    name: []const u8,
    type_name: []const u8,
    optional: bool = false,
};

pub fn defineCommand(name: []const u8, params: []const ParamSchema, result: []const u8) CommandSchema;
```

## Defining commands

```zig
const add_cmd = codegen.defineCommand("add", &.{
    .{ .name = "a", .type_name = "number" },
    .{ .name = "b", .type_name = "number" },
}, "number");

const search_cmd = codegen.defineCommand("search", &.{
    .{ .name = "query", .type_name = "string" },
    .{ .name = "limit", .type_name = "number", .optional = true },
}, "object");

const shutdown_cmd = codegen.defineCommand("shutdown", &.{}, "void");
```

## Generating the `.d.ts`

```zig
const ts = try codegen.generateTypeScript(allocator, &.{
    add_cmd,
    search_cmd,
    shutdown_cmd,
});
defer allocator.free(ts);
```

The `add(a, b)` command above produces the following declaration:

```typescript
// Generated by zero-native bridge codegen. Do not edit.
declare module "zero-native" {
  export function invoke(cmd: "add", args: {a: number; b: number}): Promise<number>;
  export function invoke(cmd: "search", args: {query: string; limit?: number}): Promise<object>;
  export function invoke(cmd: "shutdown", args: {}): Promise<void>;
}
```

Optional parameters emit a `?` suffix on the key. The result type is wrapped in `Promise<...>` for every command. Commands with no parameters produce `args: {}`.

## Build script integration

Call `generateTypeScript` from your build script and write the result to `src/zero-native.d.ts`. The function is small and pure — it is safe to call from any comptime-eligible context and from `addRunArtifact`-driven build steps.

See also: [Bridge](/bridge) for the runtime wire format, [Builtin Commands](/bridge/builtin-commands) for the built-in `zero-native.*` commands.
